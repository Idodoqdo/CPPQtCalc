Калькулятор. Инструкция по применению.

Арифметические операторы:

  • Сложение : a + b
  • Вычитание : a - b
  • Скобки : (a + b)
  • Умножение : a * b
  • Деление : a / b
  • Возведение в степень : a ^ b
  • Остаток от деления : a mod b
  • Унарный плюс: +a
  • Унарный минус : -a

Функции:

  • Вычисляет косинус : cos(x)
  • Вычисляет синус : sin(x)
  • Вычисляет тангенс : tan(x)
  • Вычисляет арккосинус : acos(x)
  • Вычисляет арксинус : asin(x)
  • Вычисляет арктангенс : atan(x)
  • Вычисляет квадратный корень : sqrt(x)
  • Вычисляет натуральный логарифм : ln(x)
  • Вычисляет десятичный логарифм : log(x)

Также калькулятор имеет кнопку "x", которая записывает 'x' в выражение. Вы можете посчитать выражение с "x", выбрав или введя значение в спинбоксе, и калькулятор посчитает выражение с заданным значением "x". Имеет возможность записи числа в экспоненциальном виде с помощью кноки 'e'

Если вы хотите отчистить поле ввода, то нажмите "AC"

В калькуляторе можно посроить график заданной функции с возможностью выбора области определения(D(f)) и области значения(E(f)). Для этого введите функцию, задайте соответсвующие значения в спинбоксы и нажмите кнопку "Посроить график".
